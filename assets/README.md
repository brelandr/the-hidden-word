# Plugin Assets

This folder ships in the plugin zip only as a stub (`index.php`).

**WordPress.org directory icons and banners must not be packaged in the plugin zip.**
Keep them in `docs/wordpress-org-assets/` and upload to the plugin SVN `assets/` directory (sibling of `trunk/`), not into `trunk/`.

| File | Dimensions | SVN location |
|------|------------|--------------|
| `icon-128x128.png` | 128×128 | `/assets/` |
| `icon-256x256.png` | 256×256 | `/assets/` |
| `banner-772x250.png` | 772×250 | `/assets/` |
| `banner-1544x500.png` | 1544×500 | `/assets/` |

Suggested design: open Bible with subtle light rays, navy (#1a5276) and cream palette matching the plugin CSS.
