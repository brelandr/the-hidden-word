# Plugin Assets for WordPress.org

Place the following assets in this folder before WordPress.org submission:

| File | Dimensions | Description |
|------|------------|-------------|
| `icon-128x128.png` | 128×128 | Plugin icon |
| `icon-256x256.png` | 256×256 | Plugin icon (retina) |
| `banner-772x250.png` | 772×250 | Plugin banner |
| `banner-1544x500.png` | 1544×500 | Plugin banner (retina) |

Suggested design: open Bible with subtle light rays, navy (#1a5276) and cream palette matching the plugin CSS.
